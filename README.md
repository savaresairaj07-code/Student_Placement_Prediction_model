# Student_Placement_Prediction_model

📌 Project Overview
This project aims to analyze the factors influencing student placements and build a predictive model to determine the likelihood of a student being placed based on their academic and professional profile. It leverages Exploratory Data Analysis (EDA) to uncover trends and Machine Learning to provide data-driven predictions.

📊 Key Features
Comprehensive EDA: Analysis of CGPA, internships, and technical skills vs. placement status.
Data Preprocessing: Handling missing values, encoding categorical variables, and feature scaling.
Predictive Modeling: Comparison of multiple ML algorithms (Logistic Regression, Random Forest, etc.).
Performance Metrics: Detailed evaluation using Accuracy, Precision, Recall, and F1-Score.
🛠️ Tech Stack
Language: Python
Analysis: Pandas, NumPy
Visualization: Matplotlib, Seaborn
Machine Learning: Scikit-Learn
Notebook: Jupyter / Google Colab
📂 Dataset Description
The dataset contains student information including:

Academic Scores: 10th, 12th, and Degree percentages/CGPA.
Work Experience: Previous internships or job roles.
Specialization: Field of study (e.g., Data Science, Computer Science).
Placement Status: (Target Variable) Placed / Not Placed.
📈 Exploratory Data Analysis (EDA) Highlights
Some of the key insights derived during the analysis:

CGPA Impact: Students with a CGPA > 8.0 have a significantly higher placement rate.
Internship Correlation: Having at least one internship increases placement probability by ~30%.
Skill Clusters: Specific technical certifications show a strong correlation with high-package offers.
